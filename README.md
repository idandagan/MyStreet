# MyStreet

### Development
* Run:
+ `npm i`
+ `npm start`
    + Starts Webpack Dev Server on port 8000 (run individually with `npm run start-client`)
    + Starts API Server on Port 8001 (run individually with `npm run start-server`)


### Production
* Run:
+ `npm run build`
    + Builds the UI into a /dist/ folder
+ `npm run start:prod`
